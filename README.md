# boss

> 一、Just like BOSS

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```

> 二、文件结构

```shell
		 ┌── assets      资源文件
		 ├── components  页面以及组件
	     ├── js          插件和工具
	     ├── router      路由配置
	src──├── styles      页面样式
	     ├── vuex        vuex配置，暂时未使用
	     ├── App.vue 	页面
	     └── main.js  	页面配置入口
			┌── data    页面所需静态JOSON数据
	static──├── images  页面所需静态图片

```
